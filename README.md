# 🎯 Smart Recommendation System

## 📌 Project Overview

The Smart Recommendation System is a Python-based application that suggests items based on the user's preferences. It uses simple rule-based logic to recommend movies, books, and music according to the category and interest selected by the user.

## 🚀 Features

* Takes user preferences as input
* Recommends Movies, Books, and Music
* Uses if-elif-else decision-making logic
* Menu-driven interface
* Runs continuously until the user exits
* Beginner-friendly and easy to modify

## 🛠️ Technologies Used

* Python
* VS Code

## 📂 Project Workflow

1. Display the main menu.
2. Take the user's category choice.
3. Ask for the user's preferred genre or interest.
4. Match the input using rule-based logic.
5. Display relevant recommendations.
6. Repeat until the user selects the Exit option.

## ▶️ How to Run

1. Install Python on your system.
2. Save the program as `recommendation_system.py`.
3. Open the project in VS Code.
4. Run the program using:

   ```
   python recommendation_system.py
   ```
5. Select a category and enter your preference to receive recommendations.
